# Personal dotfiles

This repository contains my personal dotfiles. It is managed using
[GNU Stow](https://www.gnu.org/software/stow/).

## Installation

Clone the repository and nstall the dotfiles using GNU Stow:

```bash
cd dotfiles

stow -t ~ sway
stow -t ~ alacritty
```

## Requirements

- alacritty
- brightnessctl
- j4-dmenu-desktop
- kanshi
- lxappearance
- pcmanfm
- playerctl
- sway
