# Personal dotfiles

This repository contains my personal dotfiles. It is managed using
[GNU Stow](https://www.gnu.org/software/stow/).

## Installation

Clone the repository and install the dotfiles using GNU Stow:

```bash
cd dotfiles

stow -t $HOME sway
stow -t $HOME alacritty
```

## Requirements

- alacritty
- brightnessctl
- fonts-firacode
- kanshi
- lxappearance
- mako-notifier
- notify-send
- pcmanfm
- pipewire
- playerctl
- sway
- swayimg
- swaylock
- xdg-desktop-portal
- xdg-desktop-portal-wlr
